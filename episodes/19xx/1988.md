# 1988 — Fall of Azure Dragon

နာရီများစွာကြာပြီးနောက် ဆရာကြီး Azure Dragon နှင့် သူ၏တပည့်နှစ်ယောက်သည် Deep Heaven City ၏ တယ်လီပို့ခန်းမသို့ ရောက်လာပြီး အစောင့်များသည် ချက်ချင်းပင် သူ့ကို နှုတ်ဆက်ရန် အမြန်ထွက်လာခဲ့ကြသည်။
"ကျွန်ုပ်တို့သည် Elder Azure Dragon အား ရိုသေလေးစားလျက်၊ အဘယ့်ကြောင့် ဤနေရာကို လာသနည်းဟု မေးရမလား၊ အစောင့်များ၏ Spatial Tempering Stage ခေါင်းဆောင်က လေးလေးစားစား မေးသည်။
Master Azure Dragon သည် Deep Heaven City ၏ အကြီးအကဲတစ်ဦးဖြစ်နေပြီဟူသောအချက်ကို သူရှင်းရှင်းလင်းလင်းသိခဲ့သည်။
"တယ်လီပို့ဖွဲ့စည်းမှုကို ချက်ချင်းအသက်သွင်းပါ၊ ငါ တစ်ခုခုလုပ်ဖို့ မြို့ကို ထွက်သွားရမယ်" ဟု ဆရာကြီး Azure Dragon က မကျေမနပ်အသံဖြင့် ညွှန်ကြားသည်။
"ဘာလဲ၊ တယ်လီပို့တင်းဖွဲ့စည်းခြင်းကို သင်အသုံးပြုရန် စီစဉ်နေပါသလား။ အကြီးအကဲများအဖွဲ့ထံမှ ခွင့်ပြုချက်ရှိမရှိ မေးရမလား။" Spatial Tempering စိုက်ပျိုးသူသည် ချီတုံချတုံဖြင့် မေးသည်။
"ဟမ်၊ ငါက မြို့ရဲ့ အကြီးအကဲတစ်ယောက်ဖြစ်ပြီး တယ်လီပို့စနစ် ဖွဲ့စည်းမှုကို အသုံးပြုဖို့ တောင်းဆိုတယ်။" Master Azure Dragon သည် အေးစက်သောအသံဖြင့် ထပ်ခါတလဲလဲပြောသည်။
"ခွင့်လွှတ်ပါ၊ Elder Azure Dragon၊ ဒါပေမယ့် အကြီးအကဲတွေက လွန်ခဲ့တဲ့ နှစ်ရက်က တင်းကျပ်တဲ့ အမိန့်ကို ချမှတ်ခဲ့ပြီး တယ်လီပို့စနစ်တွေကို အသုံးပြုလိုတဲ့ စိုက်ပျိုးသူအားလုံး အကြီးအကဲအဖွဲ့ရဲ့ ခွင့်ပြုချက် အရင်ရရမယ်လို့ ပြောပါတယ်။ ဒီအမိန့်ကို ဆန့်ကျင်ရင် ကျွန်တော်တို့အားလုံး ပြင်းထန်တဲ့ အပြစ်ပေးခံရလိမ့်မယ်၊ ကျေးဇူးပြုပြီး လုပ်သင့်လုပ်ထိုက်တာထက် ပိုခက်ခဲအောင် မလုပ်ပါနဲ့" ဟု Spattiv က ပြောသည်။
ဒါကိုကြားတော့ ဆရာကြီး Azure Dragon ရဲ့ အမူအရာက အနည်းငယ် ပြောင်းလဲသွားပြီး ခဏလောက် တွေးတောကြည့်ပြီးနောက် သူက "ကျွန်တော် မြင်ပါတယ်။ အဲဒီအခါကျရင် အကြီးအကဲအဖွဲ့ဆီကနေ ခွင့်တောင်းလိုက်မယ်" လို့ ပြောပါတယ်။
"မင်းရဲ့ ထည့်သွင်းစဉ်းစားမှုအတွက် ကျေးဇူးတင်ပါတယ်၊ Elder Azure Dragon၊ ငါတို့ တကယ်ကို ကျေးဇူးတင်မိပါတယ်။. Ah Senior Azure Dragon မင်း ဘာလုပ်နေတာလဲ။" Spatial Tempering စိုက်ပျိုးသူသည် သူ့ပတ်ဝန်းကျင်တွင် လေရုတ်တရက် တင်းကျပ်လာသောအခါတွင် သက်ပြင်းတစ်ချက်ချကာ လုံး၀ မလှုပ်ရှားနိုင်တော့ပေ။
အခြားသော အစောင့်များအားလုံးကိုလည်း မမြင်နိုင်သော ကန့်သတ်ချုပ်ချယ်မှုစွမ်းအားဖြင့် ချုပ်နှောင်ထားပြီး တုန်လှုပ်ခြောက်ခြားဖွယ် အမူအရာများ သူတို့မျက်နှာအားလုံးတွင် ပေါ်လာသည်။
"ထိတ်လန့်နေစရာ မလိုပါဘူး ငါ့မှာ လုပ်စရာတွေ ရှိတယ်၊ ဒီမှာ နှောင့်နှေးဖို့ ငါ မတတ်နိုင်ဘူး" Master Azure Dragon က အေးစက်သော အမူအရာဖြင့် ပြောသည်။
ထို့နောက် လက်တစ်ဖက်ကို မြှောက်လိုက်သည်နှင့် အဖြူရောင် ကျောက်စိမ်းတံဆိပ်တစ်ခု ရုတ်တရက် ပေါ်လာသည်။ သူသည် ခန်းမတံခါးနောက်ဘက်ရှိ ကန့်သတ်ချက်များဆီသို့ တံဆိပ်ကို ဝှေ့ယမ်းလိုက်ပြီး ကန့်သတ်ချက်များကို ပျက်ပြယ်စေရန်အတွက် အဖြူရောင်မီးတိုင်တစ်ခု ထွက်ပေါ်လာသည်။
ဒါကိုမြင်လိုက်တာနဲ့ ဆရာကြီး Azure Dragon ရဲ့ မျက်လုံးထဲမှာ ကြည်နူးမှုအရိပ်အမြွက် ပေါ်လာပြီး သူ့တပည့်တွေကို ခန်းမထဲက သေးငယ်တဲ့ တယ်လီပို့ပုံသဏ္ဍာန်တစ်ခုဆီ ချက်ချင်း ပို့ဆောင်လိုက်ပါတယ်။

ဖွဲ့စည်းမှု စတင်လိုက်သည်နှင့် တပြိုင်နက် တုန်လှုပ်ချောက်ချားသော အသံတစ်ခု ထွက်ပေါ်လာပြီး Master Azure Dragon ၏ အဖြူရောင် အလင်းတန်းများကြားတွင် ပျောက်ကွယ်သွားခဲ့သည်။
ထိုမှသာ အစောင့်များသည် ၎င်းတို့၏ လှုပ်ရှားသွားလာနိုင်မှုကို ပြန်လည်ရရှိပြီး Spatial Tempering စိုက်ပျိုးသူသည် ချက်ချင်းပင် ထိတ်လန့်တကြား အော်ဟစ်ကာ "ဒီမှာ ဖြစ်ခဲ့တာတွေကို လူကြီးတွေကို အကြောင်းကြားပြီး ကန့်သတ်ချက်တွေကို ချက်ချင်းပြင်ပေး!"
အစောင့်အချို့သည် ပျက်စီးနေသော ကန့်သတ်ချက်များကို ပြုပြင်ရန်အတွက် ရတနာများကို ဆင့်ခေါ်ကာ ဆက်သွယ်ရေးပုံစံပြားများကို အလျင်အမြန်ဆွဲထုတ်ကြသည်။
ထို့ကြောင့် တယ်လီပို့တ်ခန်းမတစ်ခုလုံး ပရမ်းပတာအခြေအနေသို့ ကျဆင်းသွားခဲ့သည်။
ဤအခိုက်အတန့်တွင် Master Azure Dragon သည် ယခုလေးတင်အသုံးပြုခဲ့သော တယ်လီပို့စနစ်မှ စူးစူးရှရှ အသံတစ်ခု ထွက်ပေါ်လာပြီး ၎င်းကို ပြန်လည်အသက်သွင်းခဲ့သည်။
ထို့နောက် ချက်ခြင်းတွင် အလွန်သေးငယ်သော အဖြူရောင်အလင်းတန်းများကြားတွင် အလွန်သေးငယ်သော ရုပ်အဆင်းသဏ္ဍာန်ကို သယ်ဆောင်သွားခဲ့ပြီး အစောင့်အကြပ် မည်သူမျှ ဖော်ထုတ်နိုင်ခြင်း မရှိခဲ့ပေ။
"အခုတလော တခြားတစ်ယောက်ယောက်က တယ်လီပို့လိုက်တာလား။" အစောင့်တစ်ယောက်က သူ့မျက်လုံးတွေက သူ့ကိုလှည့်စားတာလားလို့ မသေချာတဲ့ပုံစံနဲ့ မေးတယ်။
ထိုအချိန်တွင် ဆရာကြီး Azure Dragon နှင့် သူ၏ တပည့် နှစ်ယောက်သည် တောင်ကြီး၏ ဝမ်းဗိုက်တွင် တည်ဆောက်ထားသော bluestone လမ်းကြောင်းတစ်ခုကို ဖြတ်၍ ပျံသန်းလာကြသည်။ ခဏကြာပြီးနောက် သူတို့သည် ကန့်သတ်မှုဖြင့် ဖုံးကွယ်ထားသည့် လိုဏ်ဂူတစ်ခုမှ ထွက်လာကြပြီး အချို့သော ဦးတည်ရာဆီသို့ ပျံသန်းသွားကြသည်။
သူတို့ ထွက်သွားပြီး သိပ်မကြာခင်မှာပဲ ရွှေရောင်အလင်းတန်းတွေ ဂူထဲကနေ ပြေးထွက်လာပြီး ပြင်းပြင်းထန်ထန် လိုက်ရှာခဲ့တယ်။
တစ်ရက်ခွဲကြာသောအခါ၊ Master Azure Dragon ၏ သုံးယောက်သားသည် Deep Heaven City မှ ကီလိုမီတာ မရေမတွက်နိုင်သော လွင်ပြင်တစ်ခုပေါ်တွင် ပျံသန်းနေပါသည်။
သုံးယောက်ထဲမှာ Master Azure Dragon က ရှေ့ဆုံးမှာ ပျံသန်းနေပြီး သူ့အမူအရာက အရေးကြီးတဲ့အရာတစ်ခုကို တွေးတောနေတယ်လို့ ညွှန်ပြနေပါတယ်။
အကွာအဝေး၌ အဆုံးမရှိသော သဲကန္တာရကို မြင်နိုင်သည်။
ဤအခိုက်အတန့်တွင် Master Azure Dragon ၏နောက်ပေ 100 ကျော်အကွာတွင် ပဲ့တင်ထပ်သော ပေါက်ကွဲသံနှစ်ခုထွက်ပေါ်လာပြီး သေးငယ်သောရွှေရောင်အစက်နှစ်စက်သည် လေဖိအားနည်းရပ်ဝန်းမှ ထွက်ပေါ်လာသည်။
ချွန်ထက်သော ခြေသည်းများ သည် လေထဲသို့ ပျံ့လွင့်သွားကာ Master Azure Dragon ၏ သုံးယောက်ဆီသို့ တိုက်ရိုက် ထိမှန်သွားသည်။
ကုတ်ခြစ်သံများက အာကာသအတွင်း ရွှေရောင်အက်ကွဲကြောင်းများကို ချန်ထားခဲ့ကာ ရှောင်လွှဲရန် မဖြစ်နိုင်သော ဧရာမကျော့ကွင်းကြီးထဲသို့ ခုန်ဆင်းသွားကြသည်။
Master Azure Dragon သည် ဤအရာကြောင့် အလွန်ထိတ်လန့်သွားပြီး သူ့မျက်နှာပေါ်တွင် ထိတ်လန့်သောအကြည့်တစ်ခု ပေါ်လာသည်။
