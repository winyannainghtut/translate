Azure Night Python's ခန္ဓာကိုယ်သည် မည်မျှထူနေသော်လည်း၊ ထိုမေးရိုးကြီးများက ၎င်းကို ဖိထားမည်ဆိုလျှင်၊ ၎င်းကို နှစ်ခြမ်းကွဲသွားလိမ့်မည်မှာ သေချာပါသည်။

သို့သော်လည်း ရူးသွပ်သွားပြီးနောက်၊ Azure Night Python's သည် Dark Beast ၏ မြန်နှုန်းထက် မနိမ့်ကျဘဲ ၎င်း၏ ပါးစပ်ပေါက်ကို မဖွင့်မီ ခေါင်းကို ဆတ်ခနဲ ဆုတ်ခွာလိုက်ပြီး Dark Beast ဆီသို့ လှည့်သွားခဲ့သည်။

ဤအခိုက်အတန့်တွင်၊ ၎င်း၏မူလပစ်မှတ်၏ကိုယ်ထည်၊ အခြား Dark Beast ဆွဲငင်အားသည် Azure Night Python ဆီသို့ တိုက်ရိုက်ထိသွားသဖြင့် မယုံနိုင်လောက်အောင် ကျယ်ပြန့်သွားခဲ့သည်။

ထို့ကြောင့် သားရဲနှစ်ကောင်သည် ကျယ်လောင်စွာ ပေါက်ကွဲသံများကြားတွင် ခိုက်မိသွားကာ ၎င်းတို့သည် အတော်လေး လိုက်ဖက်ညီပုံရသည်။

သူတို့နှစ်ဦးကို မသိလိုက်ဘဲ၊ ပေ ၂၀ ပတ်လည် အချင်းဝက်ရှိ ကြီးမားသော အပေါက်တစ်ခုကို ၎င်းတို့နှင့် ၁၀ ကီလိုမီတာအကွာတွင်ရှိသော သာမန်ပုံသဏ္ဍာန်ရှိသော ဧရာမသစ်ပင်ကြီးတစ်ခုအဖြစ် ထွင်းထုထားသည်။

သစ်ပင်၏အပေါက်အတွင်း၌ တင်ပျဉ်ခွေပုံစံဖြင့် အထဲမှာ လူသုံးယောက်ထိုင်နေပြီး လေထုအလယ်တွင် ပျံဝဲနေသော မှန်တစ်ချပ်ကို အကဲဖြတ်ကာ သူတို့၏မျက်နှာများကို စူးစူးစိုက်စိုက်ကြည့်နေကြသည်။

မှန်၏နောက်ဘက်သည် ငွေရောင်အလင်းတန်းများဖြင့် တောက်ပနေပြီး၊ မျက်နှာပြင်၏မှန်သည် အလွန်ကြည်လင်ပြတ်သားသော ပုံရိပ်ကို ပြသနေပါသည်။ ပုံသည် Dark Beast နှင့် Azure Night Python အကြားတိုက်ပွဲမှလွဲ၍ အခြားမဟုတ်ပေ။

အကွာအဝေးရှိ မြင်ကွင်းအားလုံးကို မှန်ထဲတွင် ပုံတူကူးထားပြီး၊ အရာအားလုံးသည် ၎င်းတို့၏ တကယ့်အရွယ်အစားထက် အဆမရေမတွက်နိုင်သော သေးငယ်သည်မှတပါး၊

ဒီသုံးယောက်ထဲမှာ တစ်ယောက်က အပြာရောင်ဝတ်ထားတဲ့ အမျိုးသား၊ တစ်ယောက်က သူ့ခေါင်းထက်မှာ အဖြူထည်ဝတ်ထားတဲ့ အမျိုးသမီးတစ်ယောက်ဖြစ်ပြီး နောက်ဆုံးကတော့ ပိန်ပိန်ပါးပါး အသားဝါတဲ့ အမျိုးသားတစ်ယောက်ပါ။

၎င်းတို့မှာ Han Li၊ Liu Shui'er နှင့် Shi Kun တို့မှလွဲ၍ အခြားမဟုတ်၊ တစ်လနီးပါးကြာပြီးနောက် ဤနေရာသို့ နောက်ဆုံးရောက်ရှိခဲ့ကြသည်။

သူတို့ သုံးယောက်စလုံး မှိတ်တုတ်မှိတ်တုတ် မမှိတ်မသုန် မျက်လုံးများဖြင့် မှန်ကို စိုက်ကြည့်ရင်း ညစ်ညမ်းသော အမူအရာများ ဝတ်ဆင်ထားကြသည်။

တစ်နာရီနီးပါးကြာပြီးနောက်၊ ပုံထဲက Dark Beast သည် ရာနှင့်ချီသော လင်းနို့နက်များအဖြစ်သို့ ကွဲသွားခဲ့ပြီး နောက်ဆုံးတွင် ဒဏ်ရာရသွားသော Azure Night Python ကို သေဆုံးသည်အထိ ဝတ်ဆင်ကာ ၎င်း၏သားကောင်၏အူတိုင်ကို ကိုက်စားကာ စိတ်ဓာတ်ပြင်းထန်စွာ ထွက်ခွာသွားခဲ့သည်။

Liu Shui'er ဂါထာတံဆိပ်ကို မှန်ပေါ်ပစ်ချလိုက်ပြီး သက်ပြင်းချရင်း ပုံရိပ်က ပျောက်ကွယ်သွားပြီး "မင်းတို့ကိုယ်တိုင် သက်သေခံပြီးရင် ဒါတွေကို မင်းဘယ်လိုထင်လဲ"

အမှားအယွင်းများတွေ့ပါက (ကျိုးပြတ်နေသောလင့်ခ်များ၊ စံမဟုတ်သောအကြောင်းအရာများ စသည်ဖြင့်) ) ကျေးဇူးပြု၍ ကျွန်ုပ်တို့အား အမြန်ဆုံးပြုပြင်နိုင်စေရန် အသိပေးပါ။

အကြံပြုချက်- အခန်းများကြားတွင် ရှာဖွေရန် ဘယ်၊ ညာ၊ A နှင့် D ကီးဘုတ်ခလုတ်များကို သင်သုံးနိုင်သည်။
