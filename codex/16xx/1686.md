Azure Night Python ၏ကိုယ်ထည်သည် မည်မျှထူနေသော်လည်း ထိုမေးရိုးကြီးများက ၎င်းကို တွယ်ကပ်လိုက်လျှင် ၎င်းကို နှစ်ခြမ်းကွဲသွားသည်မှာ သေချာပါသည်။

သို့သော်လည်း ရူးသွပ်သွားပြီးနောက်၊ Azure Night Python ၏ အရှိန်သည် Dark Beast ထက် မနိမ့်ကျဘဲ ဆတ်ခနဲ ခေါင်းမှ ဆုတ်သွားကာ ၎င်း၏ ပါးစပ်ကို ဖွင့်ကာ Dark Beast ဘက်သို့ လှည့်သွားသည်။

ဤအခိုက်အတန့်တွင်၊ ၎င်း၏မူလပစ်မှတ်၏ကိုယ်ထည်၊ အခြား Dark Beast ပရိုဂျက်တာသည် Azure Night Python ဆီသို့ တိုက်ရိုက်အခိုက်အတန့်ဖြစ်ပြီး မယုံနိုင်လောက်စရာအတိုင်းအတာအထိ ရုတ်တရက် ရှည်လျားသွားသည်။

ထို့ကြောင့် သားရဲနှစ်ကောင်သည် ကျယ်လောင်စွာ ပေါက်ကွဲသံများကြားတွင် ခိုက်မိသွားကာ ၎င်းတို့သည် အတော်လေး လိုက်ဖက်ညီပုံရသည်။

သူတို့နှစ်ဦးကို မသိလိုက်ဘဲ၊ ပေ ၂၀ ပတ်လည် အချင်းဝက်ရှိ ကြီးမားသော အပေါက်တစ်ခုကို ၎င်းတို့နှင့် ၁၀ ကီလိုမီတာအကွာတွင်ရှိသော သာမန်ပုံသဏ္ဍာန်ရှိသော ဧရာမသစ်ပင်ကြီးတစ်ခုအဖြစ် ထွင်းထုထားသည်။

သစ်ပင်၏အပေါက်အတွင်း၌ တင်ပျဉ်ခွေပုံစံဖြင့် အထဲမှာ လူသုံးယောက်ထိုင်နေပြီး လေထုအလယ်တွင် ပျံဝဲနေသော မှန်တစ်ချပ်ကို အကဲဖြတ်ကာ သူတို့၏မျက်နှာများကို စူးစူးစိုက်စိုက်ကြည့်နေကြသည်။

မှန်၏နောက်ဘက်သည် ငွေရောင်အလင်းတန်းများဖြင့် တောက်ပနေပြီး၊ မျက်နှာပြင်၏မှန်သည် အလွန်ကြည်လင်ပြတ်သားသော ပုံရိပ်ကို ပြသနေပါသည်။ ပုံသည် Dark Beast နှင့် Azure Night Python အကြား တိုက်ပွဲမှလွဲ၍ အခြားမဟုတ်ပေ။

All အကွာအဝေးရှိ မြင်ကွင်းများကို မှန်ထဲတွင် စုံလင်စွာ ပုံတူကူးထားပြီး၊ အရာအားလုံးသည် ၎င်းတို့၏ တကယ့်အရွယ်အစားထက် အဆမရေမတွက်နိုင်သော သေးငယ်သည်မှတပါး၊

ဒီသုံးယောက်ထဲမှာ တစ်ယောက်က အပြာရောင်ဝတ်ထားတဲ့ အမျိုးသား၊ တစ်ယောက်က သူ့ခေါင်းထက်မှာ အဖြူထည်ဝတ်ထားတဲ့ အမျိုးသမီးတစ်ယောက်ဖြစ်ပြီး နောက်ဆုံးကတော့ ပိန်ပိန်ပါးပါး အသားဝါတဲ့ အမျိုးသားတစ်ယောက်ပါ။

၎င်းတို့မှာ Han Li၊ Liu Shui'er နှင့် Shi Kun တို့မှလွဲ၍ အခြားမဟုတ်၊ တစ်လနီးပါးကြာပြီးနောက် ဤနေရာသို့ နောက်ဆုံးရောက်ရှိခဲ့ကြသည်။

All သူတို့ သုံးယောက်သည် မှိတ်တုတ်မှိတ်တုတ် ပြင်းထန်သော မျက်လုံးများဖြင့် မှန်ကို စိုက်ကြည့်ရင်း ဆိုးရွားသော အမူအရာများကို ၀တ်ဆင်ထားသည်။

တစ်နာရီနီးပါးကြာပြီးနောက်၊ ပုံပါ Dark Beast သည် ရာနှင့်ချီသော လင်းနို့နက်များအဖြစ်သို့ ကွဲထွက်သွားပြီး နောက်ဆုံးတွင် ဒဏ်ရာရနေသော Azure Night Python ကို သေဆုံးသည်အထိ ဝတ်ဆင်ကာ ၎င်း၏သားကောင်၏အူတိုင်ကို ကိုက်စားကာ စိတ်ဓာတ်ပြင်းထန်စွာ ထွက်ခွာသွားခဲ့သည်။

Liu Shui မင်းက မှန်ပေါ်ကို ဂါထာတံ ဆိပ်ချလိုက်ပြီး သက်ပြင်းချရင်း ရုပ်ပုံက ပျောက်သွားတယ်၊ "What မင်းကိုယ်တိုင် သက်သေခံပြီးရင် ဒါတွေကို မင်းထင်နေလား။"
